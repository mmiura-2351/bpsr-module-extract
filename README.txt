Module OCR Tool かんたん使い方
==============================

このファイルは zip 同梱用の簡易説明です。

同梱ファイル
------------
- ModuleOcrTool.exe
- ocr_range_guide.png（範囲設定の例）

使い方
------
1. `ModuleOcrTool.exe` を起動する
2. ゲーム画面でモジュール詳細を表示する
3. OCR取得範囲を設定する（`モジュール名 / カテゴリ名 / 効果1 / 効果2 / 効果3`）
4. `処理開始` を押す
5. `OCR実行` を押す
6. OCR結果画面で必要なら修正して `確定` を押す
7. 必要な数だけ 5〜6 を繰り返す
8. 最後に `JSON出力(新規)` または `既存JSON更新` を押す

うまく読めないとき
------------------
- `ocr_range_guide.png` を見ながら範囲を取り直す
- 使わない効果枠はチェックをOFFにする
- 枠を少し狭くして再実行する
